\# Project 1A — S3 Static Website Hosting



\## What I Built

A personal portfolio website hosted entirely on Amazon S3 

using Static Website Hosting — no web server required.



\## AWS Services Used

\- Amazon S3 (Simple Storage Service)

\- S3 Static Website Hosting

\- S3 Bucket Policy for public access



\## Steps Completed

1\. Created S3 bucket with public access enabled

2\. Uploaded index.html portfolio page

3\. Configured Static Website Hosting

4\. Added bucket policy for public read access

5\. Verified website live via S3 endpoint URL



\## Live URL

http://indira-cloud-portfolio.s3-website-us-east-1.amazonaws.com



\## Key Learnings

\- S3 can host websites without any web server

\- Bucket policies control who can access objects

\- Static website hosting requires public access + bucket policy
